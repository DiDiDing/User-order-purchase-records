这是一组有关用户购物行为的数据，包括用户的个人信息、购买的产品及其特征、订单详情等。这类数据通常用于电子商务分析、用户行为研究、市场营销决策等，字段解释如下：
年龄 (age): 用户的年龄，通常用于分析不同年龄段的用户行为。
性别 (gender): 用户的性别，可用于性别相关的市场分析。
类型 (genres): 产品或内容的类型，这里是电影类型（如喜剧、浪漫等），可以用于分析用户对不同类型内容的偏好。
评分 (rating): 用户对产品或内容的评分，通常在1到5之间，用于衡量用户满意度。
标签 (tag): 可能与产品或内容相关的关键词（如演员名），可以用于更深入的分析或推荐系统。
用户ID (user_id): 每个用户的唯一标识符，用于跟踪用户行为和分析。
订单ID (order_id): 每个订单的唯一标识符，有助于识别每次购买行为。
商品数量 (num_of_item): 每个订单中包含的商品数量。
是否退货 (returned): 一个二元变量，指示商品是否被退回。
产品ID (product_id): 每个产品的唯一标识符。
销售价格 (sale_price): 商品的销售价格。
名称 (name): 产品的名称。
类别 (category): 产品所属的类别。
零售价格 (retail_price): 商品的原始零售价格。
订单金额 (order_amount): 订单的总金额。
订单年月 (order_year_month): 订单发生的年月，便于时间序列分析。
